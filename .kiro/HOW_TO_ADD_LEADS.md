## How to Add a New Lead (Step by Step)

### Step 1: Click "Add New Lead" Button
- Go to **Leads** page
- Click the **"+ Add New Lead"** button (top right)
- A form appears

### Step 2: Fill the Form

**Fill these fields:**

1. **Name** - Customer name
   - Example: "Hotel Sunrise"

2. **Phone** - Customer phone number
   - Example: "9876543212"

3. **Address** - Service address
   - Example: "Beach Road, Calicut"

4. **Services** - Add services (you can add multiple)
   - Type service name: "Pest Control"
   - Click "Add" button
   - Service appears as a tag
   - Add more services if needed
   - Example: "Pest Control", "Fumigation", "Termite Treatment"

5. **Notes** (Optional) - Any additional notes
   - Example: "Prefer morning slots"

### Step 3: Click "Save Lead"

- Click the **"Save Lead"** button
- You should see a **green message**: ✓ "Lead created successfully!"
- The form closes
- Your new lead appears in the table!

### Step 4: Verify the Lead

- Look at the **Leads** table
- Your new lead should be there with:
  - Name
  - Phone
  - Address
  - Services count
  - Status: "New"

## Tips

### Adding Services
- Type service name
- Press Enter OR click "Add" button
- Service appears as a tag
- Click X on tag to remove it
- Add as many services as you need

### Validation
- All fields are required (Name, Phone, Address, Services)
- At least one service must be added
- If you miss something, you'll see an error message

### Refresh Page
- After adding a lead, refresh the page (F5)
- Your lead will **STILL BE THERE** ✅
- Data is saved automatically!

## Common Issues

### Issue: "Save Lead" button doesn't work
**Solution**: Make sure you filled all fields:
- Name ✓
- Phone ✓
- Address ✓
- At least one service ✓

### Issue: Service doesn't get added
**Solution**: 
- Type service name
- Click "Add" button (or press Enter)
- Service should appear as a tag

### Issue: Lead doesn't appear in table
**Solution**:
- Refresh page (F5)
- Lead should appear
- Check if you're on the right page
